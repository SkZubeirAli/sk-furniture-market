# SK Furniture Market

A modern furniture e-commerce website built using **WordPress** and **WooCommerce**.

## Project Overview

This project is being developed for a furniture business with a clean, responsive, and user-friendly design.

## Features

- Responsive Design
- WooCommerce Store
- Product Categories
- Wishlist
- Shopping Cart
- Secure Checkout
- Payment Gateway Integration (Razorpay)
- Blog
- Contact Form
- SEO Optimized
- Admin Dashboard

## Tech Stack

- WordPress
- WooCommerce
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
